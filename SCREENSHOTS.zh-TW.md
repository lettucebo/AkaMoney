## 登入頁面

**路由：** `/login`

![登入頁面](docs/screenshots/login.png)

### 功能：
- **Microsoft Entra ID 整合**：單一登入 (SSO)
- **簡潔設計**：具有 AkaMoney 品牌的極簡介面
- **安全驗證**：使用 Microsoft 驗證流程

### 介面元素：
```
┌─────────────────────────────────────────┐
│                                         │
│           🔗 AkaMoney                   │
│       網址縮短服務                       │
│                                         │
│   ┌───────────────────────────────┐    │
│   │  使用 Microsoft 登入           │    │
│   │  [Microsoft 標誌]             │    │
│   └───────────────────────────────┘    │
│                                         │
│   由 Microsoft Entra ID 提供           │
│   安全登入                              │
│                                         │
└─────────────────────────────────────────┘
```

### 使用者流程：
1. 使用者點擊「使用 Microsoft 登入」
2. 重定向到 Microsoft 登入頁面
3. 使用 Microsoft 憑證進行驗證
4. 成功後重定向回儀表板

## 網址管理

### 建立網址視窗

**觸發方式：** 點擊「+ 建立新網址」按鈕

![建立視窗](docs/screenshots/create-modal.png)

### 介面：
```
┌─────────────────────────────────────────┐
│ 建立短網址                          [×] │
├─────────────────────────────────────────┤
│                                         │
│ 原始網址 *                              │
│ ┌─────────────────────────────────────┐ │
│ │ https://                            │ │
│ └─────────────────────────────────────┘ │
│                                         │
│ 自訂短代碼                              │
│ ┌─────────────────────────────────────┐ │
│ │ my-custom-code                      │ │
│ └─────────────────────────────────────┘ │
│ 3-20 個字元 (a-z, 0-9, -, _)           │
│                                         │
│ 標題                                    │
│ ┌─────────────────────────────────────┐ │
│ │ 連結的可選標題                       │ │
│ └─────────────────────────────────────┘ │
│                                         │
│              [取消]  [建立]             │
│                                         │
└─────────────────────────────────────────┘
```

### 表單欄位：

1. **原始網址**（必填）
   - 輸入類型：URL
   - 驗證：必須是有效的 HTTP/HTTPS 網址
   - 佔位符：「https://example.com/your-long-url」

2. **自訂短代碼**（選填）
   - 輸入類型：文字
   - 模式：`[a-zA-Z0-9-_]{3,20}`
   - 驗證： 
     - 3-20 個字元
     - 僅限英數字元、連字號、底線
     - 必須唯一
   - 如未提供則自動生成

3. **標題**（選填）
   - 輸入類型：文字
   - 最大長度：255 個字元
   - 用途：幫助識別連結

### 驗證訊息：
```
❌ 無效的網址格式
❌ 短代碼已存在。請選擇其他代碼。
❌ 無效的短代碼格式。使用 3-20 個英數字元。
✅ 短網址建立成功！
```

### 成功狀態：
建立成功後：
- 視窗自動關閉
- 新網址出現在列表頂部
- 成功訊息（可選的彈出通知）

## 刪除確認

**觸發方式：** 點擊網址上的刪除按鈕

![刪除視窗](docs/screenshots/delete-modal.png)

### 介面：
```
┌─────────────────────────────────────────┐
│ 確認刪除                            [×] │
├─────────────────────────────────────────┤
│                                         │
│  您確定要刪除此網址嗎？                 │
│                                         │
│  此操作無法復原。                       │
│                                         │
│              [取消]  [刪除]             │
│                                         │
└─────────────────────────────────────────┘
```

### 功能：
- **警告訊息**：清楚說明操作
- **非阻塞式**：視窗覆蓋層，不會中斷其他操作
- **樣式化按鈕**： 
  - 取消（次要）- 灰色
  - 刪除（危險）- 紅色
- **需要確認**：防止意外刪除

### 使用者流程：
1. 使用者點擊網址卡片上的刪除按鈕
2. 確認視窗出現
3. 使用者確認或取消
4. 如果確認：
   - 從資料庫刪除網址
   - 從列表中移除卡片
   - 刪除分析資料（CASCADE）
   - 顯示成功回饋
